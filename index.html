Установка

Загрузить Bootstrap, Popper.js первой версии для bootstrap версии 4.5.3 и ниже Popper.js 1.16.1 и Jquery

Внутри тега <head> инициализировать css стили bootstrap:

            
                <link rel="stylesheet" href="bootstrap.min.css">

Перед загрывающимся тегом </body> инициализировать jquery, popper и bootstrap:

            
                <script src="jquery-3.5.1.min.js"></script>
                <script src="popper.min.js"></script>
                <script src="bootstrap.min.js"></script>

Tooltip от popper.js

Применение

В tooltip можно указать текущий день, праздник, другую необходимую информацию для отображения при наведении курсора на день.

Описание

Указание текущей даты, типа дня, праздника и другой информации при наведение курсора на день в календаре, происходит через аттрибут data-toggle, data-placement и title.

Возможно так же изменить фон подсказок, создав свой шаблон в JS, например синий фон для tooltip

JS код

            
                $('[data-toggle="tooltip-primary"]').tooltip({
                    template: '<div class="tooltip" role="tooltip">
                                    <div class="arrow color-primary"></div>
                                    <div class="tooltip-inner bg-primary"></div>
                                </div>'
                });

Для использования достаточно указать data-toggle="tooltip-primary"

Tooltip с primary фоном

Типы дней в календаре

Рабочий день

Описание

progress__workday - CSS класс отображающий будний день в календаре.

Отображение

HTML код

                
                    <div class="progress-bar progress__date progress__workday" 
                        role="progressbar" 
                        data-toggle="tooltip" 
                        data-placement="top" 
                        title="1 Декабря" 
                        style="--daysInMonth: 31;">
                    </div>

CSS стиль

                
                    .progress__date.progress__workday {
                        background-color: var(--primary);
                    }

Текущий день

Описание

progress__currentday - CSS класс, работает с любым типом дней в progress bar. Изменяющий прозрачность этого и находящиеся перед ним днями.

Отображение

HTML код

                
                    <div class="progress-bar progress__date progress__workday progress__currentday" 
                        role="progressbar" 
                        data-toggle="tooltip" 
                        data-placement="top" 
                        title="5 Декабря" 
                        style="--daysInMonth: 31;">
                    </div>

CSS стиль

                
                    .progress__date.progress__currentday {
                        opacity: 0.9;
                    }
